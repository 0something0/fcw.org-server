<h2>Keyboard Shortcuts for Unit Orders:</h2>
  a: (a)uto-settler  (settler/worker units)
  b: (b)uild city  (settler units)
  c: (c)enter map on active unit
  d: (d)o - select unit execution
  E: build airbas(E)
  f: (f)ortify unit  (military units)
  F: build (F)ortress  (settler/worker units)
  g: (g)o to tile  (then left-click mouse to select target tile)
  h: set unit's (h)omecity  (to city on current tile)
  i: build (i)rrigation or convert terrain  (settler/worker units)
  j: no orders - skip orders for this unit this turn 
  J: cancel orders - clear current orders for this unit 
  l: (l)oad unit as cargo onto transporting unit
  m: build (m)ine or convert terrain  (settler/worker units)
  n: clean (n)uclear fallout
  N: explode (N)uclear
  o: transf(o)rm terrain  (engineer unit)
  p: clean (p)ollution  (settler/worker units)
  p: (p)aradrop (paratroopers only)
  P: (P)illage  (destroy terrain alteration)
  r: build (r)oad/railroad  (settler/worker units)
  s: (s)entry unit
  S: un(S)entry all units on tile
  L: unit go (t)o/airlift to city
  u: (u)nload unit from transporter
  U: (u)nload, show, and activate all cargo units on transport 
  v: select all units on same tile 
  V: select all units of same TYPE on tile 
alt-v: select all units NOT of this type on tile 
  x: unit auto e(x)plore
  X: select all units of same TYPE globally
  w: (w)ait: focus on next unit
ESC: Exit GOTO mode/Drag mode/Drag

<h2>Unit Movement:</h2> Numpad, Cursors:
  1: move south-west (alt-.)
  2: move south (alt-,)
  3: move south-east (alt-m)
  4: move west (alt-j)
  6: move east (alt-l)
  7: move north-west (alt-u)
  8: move north (alt-i)
  9: move north-east (alt-o)
double-tap drag: move unit 


<h2>Other Keyboard Shortcuts:</h2>
 w: return to map: close tab/window/pop-up (also F1)
f2: government tab 
f3: nations tab (players)
f6: tech tab (research)
f11: prefs tab (settings/options)
f12: docs tab (help)
space: de-select units, exit GOTO mode
shift-space: re-focus to previous map position 
alt-c: show cities tab (also F4)
shift-c: select all same type units on (c)ontinent
ctrl-c:  city labels on/off
ctrl-g: map grid on/off
ctrl-l:  show airlift counter on cities 
shift-R: revolution: choose new government 
alt-s: fullscreen mode (also F11)
ctrl-s: save game
shift-T: change tax rates
shift-Return: Turn done


<h2>Main Map (Mouse):</h2>
  Left-click on city:            Pop up city dialog
  Left-click on unit:            Select a single unit
Shift-click unit:                Add unit(s) on tile or panel to selection
Shift-click specialist:          Change ALL specialists of this type 
Ctrl-click specialist:           Cycle through primary 3 specialists (not secondary specialists)

  Right-click:                   Center tile in view
  Right-click unit:              Unit Orders Menu (pop-up)
  Right-click-and-drag:          Select units by area

  Middle-click:                  Show tile info  
  Alt-right click                Show tile info (if mouse has no middle click)
  Double-tap tile:               Show tile info (mobile/touch device only)

  Double-tap map-and-drag:       Move the map 
  Double-click:                  Add production selection to city worklist / queue 
Ctrl-double-click:               Change current city production to already highlighted production selection
